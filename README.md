PROJECT GUIDE

Frontend
	• Framework: React (Vite)
	• Editor: Visual Studio Code
	• Project: learningplatformreact
Backend (API)
	• Framework: .NET
	• Editor: Visual Studio
	• Project: LearningPlatform
Database
	• System: Microsoft SQL Server
	• Database: LearningPlatform_V2


HOW TO RUN THE PROJECT
	1. Open the backend project in Visual Studio.
	2. Run the project (click the green ▶ Start button).
→ This starts the API and opens Swagger in your browser.
	3. When the API is running, open the frontend project in Visual Studio Code.
	4. In the VS Code terminal, run:
npm run dev
	5. Open the shown localhost URL.
(You should now see the website with separate pages for Teachers, Participants, Courses, etc. and )


IMPORTANT NOTES
→ The backend must be running before starting the frontend.
→ The frontend communicates with the API using localhost endpoints.
